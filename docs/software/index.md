# Software

## Overview

The MAO Research Group develops research software to support:

- simulation-based design optimization  
- dimensionality reduction  
- surrogate modeling  
- reproducible computational workflows  

All software is designed with a strong focus on **scientific rigor, transparency, and reproducibility**.

---

## Available Software

### PME-toolkit

Toolkit for parametric model embedding and related methods for design-space dimensionality reduction.

👉 [Open PME-toolkit page](pme-toolkit.md)

---

## Software Principles

Our software development follows these principles:

- scientific transparency  
- reproducibility  
- versioned releases  
- maintainable documentation  
- clear separation between stable and experimental features
