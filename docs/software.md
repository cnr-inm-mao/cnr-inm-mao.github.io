# Software

## Overview

The group develops research software to support reproducible engineering design, dimensionality reduction, and simulation-based optimization workflows.

## Available Software

### PME-toolkit

Toolkit for parametric model embedding and related methods for design-space dimensionality reduction in engineering shape optimization.

**Main features**

- reduced-order representation of design spaces
- analytical backmapping
- support for physics-informed variants
- MATLAB and Python-oriented development
